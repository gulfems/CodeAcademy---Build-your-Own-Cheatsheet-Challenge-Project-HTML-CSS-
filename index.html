HTML_1 :Introduction to HTML

HTML Basics

HTML = HyperText Markup Language

HTML elements canonically contain opening and closing tags with raw text or other HTML tags between them.

HTML elements can be nested. The enclosed element is regarded as the child of the enclosing parent element.

Any visible content should come in <body> tags (including footer).

<h1>, <h2>, <h3>, <h4>, <h5>, <h6> are used to provide titles for content.

<p> plain text and paragraphs.

<span> – for marking a particular component and/or components in other elements.

<div> – for divisions.

<br> – line break

<img> – images

<video> – videos

<audio> – audios

<ol> – ordered list

<ul> – unordered list

<em> – for making the content italic.

<strong> – to make the content appear in bold.

HTML_2 :HTML Tags

HTML tags are generally used to describe the content that they surround. The first line of a HTML document involves <!DOCTYPE html>. This tag is regarded as a declaration and it must always appear on the first line in your HTML files. The first line which involves this declaration lets the browser know what version of HTML to expect.

All of your HTML code should be written in <html> element. The information about the webpage such as the title, should be contained by <head> of the page. The <title> of the webpage appears in a browsers’ tab.

<a> tags are quite useful as they function to link not only internal pages but also external pages or content on the same page. <!-- comment --> is used to add comments on a HTML document.

HTML_3 :Semantic HTML

Semantic elements provide info about the content. With semantic HTML tags it gets much easier to identify what is inside parent tags.

<nav> is used to define a block of navigation links such as menus and tables of contents.

<section> is used to define elements in a document, such as chapters, headings, or any other area of the document with the same theme.

<aside> is used to mark additional information that can enhance another element. Aside can involve bibliographies, endnotes, comments, pull quotes, editorial sidebars and additional information.

<figure> is used to encapsulate media such as an image, illustration, diagram, code snippet, etc.

<article> holds content that makes sense on its own such as articles, blogs, comments.

<figcaption> is used to describe the media.

<video>, <embed>, and <audio> elements are used for media files.

CSS_1 :Introduction to CSS

What is CSS?

Cascading Style Sheets or CSS is a language that is used to style the HTML document on web pages.

In order to style a document you can:

  • Create a style.css file and use a <link> to connect the stylesheet with your HTML document.
  • Use the <style></style> element to write CSS code inside the HTML document.

Classes and IDs

CSS can select HTML elements by their type, class, and ID. CSS classes and IDs have different purposes, which can affect which one you use to style HTML elements.

While classes are meant to be used many times, an ID is meant to style only one element. IDs override the styles of types and classes.

Since IDs override these styles, they should be used sparingly and only on elements that need to always appear the same.

CSS_2 :Selectors

CSS can select HTML elements by type, class, ID, and attribute.

  • All elements can be selected using the universal selector (*).
  • An element can have different states using the pseudo-class selector (e.g., :hover).
  • Multiple CSS classes can be applied to one HTML element.
  • Classes can be reusable, while IDs can only be used once.
  • IDs are more specific than classes, and classes are more specific than type. That means IDs will override any styles from a class, and classes will override any styles from a type selector.
  • Multiple selectors can be chained together to select an element. This raises the specificity but can be necessary.
  • Nested elements can be selected by separating selectors with a space (e.g., div p).
  • Multiple unrelated selectors can receive the same styles by separating the selector names with commas (e.g., h1, h2, p).
